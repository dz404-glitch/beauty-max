# Beauty Max

Beauty Max is a cute, productivity-minded social app prototype blending short-form video, visual inspiration, and AI-powered beauty advice.

## Features

- Feed for beauty and makeup posts
- Video posts and trendy look inspiration
- Commenting with a friendly in-app flow
- AI beauty bot that asks about your skin and style to generate personalized advice
- Inspiration gallery for makeup and outfits

## Run locally

1. Open the `beauty-max` directory.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local Vite URL shown in the terminal.
